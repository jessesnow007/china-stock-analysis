# 中国A股分析

分析中国 A 股市场数据，提供股票行情、技术指标和基本面分析。

## 功能

### 1. 股票行情查询
获取指定股票的实时行情数据，包括价格、涨跌幅、成交量等。

### 2. 技术指标分析
计算和分析常用技术指标：
- MACD（指数平滑异同移动平均线）
- KDJ（随机指标）
- RSI（相对强弱指标）
- MA（移动平均线）
- BOLL（布林带）

### 3. K线形态识别
识别常见K线形态：
- 锤子线 / 倒锤子线
- 吞没形态
- 早晨之星 / 黄昏之星
- 三只乌鸦
- 十字星

### 4. 基本面数据
查询公司基本面信息：
- PE（市盈率）
- PB（市净率）
- ROE（净资产收益率）
- 营收增长率
- 净利润增长率

## 数据来源

使用公开的金融数据接口获取行情数据。
